git Checkout

git checkout entre commits

O git checkout pode ser combinado com o HEAD para navegar entre commits, o comando git checkout HEAD coloca voce no commit mais recente do brancj atual, não ha mudanças percepitivas se voce ja estiver no estado mais recente, mas pode ser util se voce tiver feito alteraçoes não commitadas e quiser restaurar o diretorio de trabalho ao ultimo commmit.

Voltar para um Commit Anterior Relativo:

Com o comando git checkout HEAD~N , você retorna ao estado do repositório de N commits atrás, onde N é o número de commits que você deseja voltar.
git checkout HEAD~1 Volta um commit atrás.
git checkout HEAD~2 Volta dois commits atrás, e assim por diante.

checkout para um commit especifico

Se colocarmos git checkout "Codigo especifico de um commit" voltaremos exatamente para esse commit. Por ecxemplo : git checkout 6dcb09bcca3eabcd123456789abcde123456789a
